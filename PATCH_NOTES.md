# 패치 노트 (Patch Notes) - v1.2.0 (최신)

본 버전에서는 **노래 가사 인식 최적화**와 **멀티 모드 전환 시스템**이 도입되어, 영상의 성격에 따라 최상의 자막 품질을 얻을 수 있습니다.

## 🚀 주요 업데이트 내용

### 1. 멀티 작업 모드 (`--mode`) 도입
- **Movie 모드 (기본):** 영화, 드라마, 강의 등 대화 중심 영상에 최적화. VAD(음성 감지)를 사용하여 무음 구간을 빠르게 건너뜁니다.
- **Music 모드:** 노래 가사 추출에 특화. 배경음악(BGM) 방해를 이겨내기 위해 VAD를 끄고, `repetition_penalty`와 가사 전용 `initial_prompt`를 적용하여 환각 현상을 억제합니다.

### 2. 자동 자막 백업 (Auto-Backup) 시스템
- **덮어쓰기 방지:** 기존에 생성된 자막 파일이 있을 경우, 실행 시점에 타임스탬프를 붙여 자동으로 이름을 변경합니다. (예: `video_ja_20260304_2230.srt`)
- **버전 비교 용이:** 설정을 바꿔가며 여러 번 실행해도 이전 결과물이 보존되어 품질 비교가 쉬워졌습니다.

### 3. Whisper 엔진 극대화 (`large-v3`)
- **최고 성능 모델:** 기본 모델을 `large-v3`로 상향하여 인식률을 극대화했습니다.
- **정밀 파라미터 튜닝:** `initial_prompt`를 통해 모델에 문맥 정보를 제공하고, 무음 임계값(`no_speech_threshold`) 등을 미세 조정했습니다.

### 4. 번역 엔진 안정화 및 보안
- **DeepL SDK 최적화:** 대량의 자막을 한 번에 번역할 때 발생하던 리스트 처리 오류를 수정했습니다.
- **환경 변수(.env) 관리:** API 키 유출 방지를 위해 `.env` 파일 로드 방식을 표준화했습니다.

### 5. 오디오 추출 품질 개선
- **PCM 무압축 추출:** 오디오 추출 시 `pcm_s16le` 코덱을 명시하여 소리 뭉개짐을 방지하고 Whisper와의 호환성을 높였습니다.

# 패치 노트 (Patch Notes) - v1.1.0

본 버전에서는 사용자 편의성을 극대화하기 위한 **자동화 시스템**과 **설정 관리 기능**이 대폭 강화되었습니다.

## 🚀 주요 업데이트 내용

### 1. 자동화 및 배치 처리 (Batch Processing) 도입
- **폴더 기반 자동화:** 이제 파일을 하나씩 지정할 필요가 없습니다. `input/` 폴더에 영상을 넣고 실행하면 폴더 내 모든 영상을 자동으로 찾아 처리합니다.
- **결과물 자동 분류:** 완성된 자막은 `output/` 폴더에, 중간 작업 파일은 `work/` 폴더에 깔끔하게 분리되어 저장됩니다.

### 2. 중앙 집중식 설정 파일 (`config.py`) 추가
- **간편한 설정 변경:** 이제 코드를 건드리지 않고 `config.py` 파일만 열어서 모델 크기(`tiny`~`large-v3`), 언어, 장치 설정(GPU/CPU) 등을 한눈에 보고 바꿀 수 있습니다.
- **상세 주석 제공:** 각 설정 항목에 대한 한글 설명을 추가하여 초보자도 쉽게 커스터마이징할 수 있습니다.

### 3. 번역 안정성 및 효율성 개선
- **글자 수 제한 안전장치:** Google 번역 API의 글자 수 제한(약 4,500자)을 체크하는 로직을 추가하여 대용량 자막 처리 시 발생하던 끊김 현상을 방지했습니다.
- **지능형 배칭:** 줄 수와 글자 수를 동시에 고려하여 번역 속도를 최적화했습니다.

### 4. 하드웨어 제어 옵션 강화
- **GPU/CPU 선택 기능:** `config.py`에서 `DEVICE` 설정을 통해 그래픽카드(CUDA) 사용 여부를 강제하거나 자동으로 선택할 수 있습니다.
- **연산 정밀도(Compute Type) 조절:** 장치에 맞는 최적의 정밀도(`float16`, `int8`)를 자동으로 선택하여 연산 효율을 높였습니다.

### 5. 기타 개선 사항
- **중복 작업 건너뛰기:** 이미 오디오가 추출된 경우 해당 단계를 건너뛰어 전체 처리 시간을 단축했습니다.
- **임시 파일 관리:** 작업 완료 후 `work/` 폴더의 임시 파일을 자동으로 삭제하는 옵션을 추가했습니다.

## 🛠 사용 방법
1. `input/` 폴더에 영상 파일을 넣습니다.
2. (선택사항) `config.py`에서 원하는 모델이나 설정을 확인합니다.
3. `python video_subtitles/main.py`를 실행합니다.
4. `output/` 폴더에서 완성된 자막을 확인합니다.
